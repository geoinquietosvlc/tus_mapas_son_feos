Introducción
============

*Caveat Emptor*::

    Este es un taller **especulativo,** no científico.

    No hay una receta para hacer las cosas bien, no hay una bala de plata.


Habría que tener claros unos conceptos

* Diseño cartográfico >> Arte cartográfico >> Estética cartográfica

* Útil / no útil **¿Comuníca?**

* Feo / Bonito **¿Es estético?**

Hay dos componentes diferenciadas:

* El objetivo (Mensaje / Qué)

* El lenguaje (Diseño / Cómo)

75% de las veces (estadísitca inventada) no sabemos ni el qué ni el cómo de un
mapa... y estos acaban siendo feos.

Existe una alta probabilidad de que al final, tus mapas sigan siendo feos...
espero que por lo menos sepas por qué.
